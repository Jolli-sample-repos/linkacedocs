# LinkAce Overview

LinkAce is a powerful, self-hosted bookmark manager and archive solution designed to help users organize, manage, and preserve their web bookmarks. Built with PHP and Laravel, LinkAce offers a modern alternative to traditional bookmarking services with a focus on privacy, control, and comprehensive link management.

## What is LinkAce?

LinkAce is an open-source application that allows you to save, categorize, and search through your bookmarks efficiently. Unlike cloud-based bookmarking services, LinkAce gives you complete control over your data by running on your own server or hosting environment. It's designed for users who want to maintain a personal knowledge base of web resources without relying on third-party services.

The project was created by Kovah and is actively maintained on GitHub, with a growing community of contributors and users who value privacy and data ownership.

## Key Features

### Bookmark Management
- **Save and organize links** with titles, descriptions, and custom metadata
- **Automatic metadata extraction** from web pages including titles, descriptions, and favicons
- **Duplicate detection** to prevent saving the same link multiple times
- **Bulk operations** for managing multiple bookmarks at once

### Organization and Categorization
- **Lists** for grouping related bookmarks together
- **Tags** for flexible, multi-dimensional organization
- **Search functionality** with full-text search capabilities
- **Filtering options** to quickly find specific bookmarks

### Link Archiving
- **Internet Archive integration** to create permanent snapshots of web pages
- **Automatic backups** of link content to prevent loss from dead links
- **Private backups** option for storing page content locally

### Privacy and Security
- **Self-hosted** - complete control over your data
- **User authentication** with support for multiple users
- **Guest access** option for sharing bookmarks publicly or privately
- **API access** for programmatic bookmark management

### Import and Export
- **Import from popular services** including Pocket, Instapaper, and browser bookmarks
- **Export functionality** to back up your bookmarks
- **API endpoints** for integration with other tools and services

## Technical Stack

LinkAce is built using modern web technologies:

- **Backend**: PHP 8.x with Laravel framework
- **Database**: MySQL/MariaDB or PostgreSQL
- **Frontend**: Vue.js with Tailwind CSS
- **Containerization**: Docker support for easy deployment
- **Queue System**: Laravel queues for background processing

## Installation Options

LinkAce offers multiple deployment methods to suit different user preferences and technical expertise:

### Docker (Recommended)
The easiest way to get started with LinkAce is through Docker. The project provides official Docker images that include all necessary dependencies and configurations. This method is ideal for users who want a quick setup with minimal configuration.

### Manual Installation
For users who prefer more control or need to integrate LinkAce into existing server infrastructure, manual installation is supported. This requires a web server (Apache/Nginx), PHP, and a compatible database.

### Hosting Requirements
- PHP 8.0 or higher
- MySQL 5.7+ / MariaDB 10.3+ or PostgreSQL 10+
- 512 MB RAM minimum (1 GB recommended)
- Modern web browser for the interface

## Use Cases

LinkAce is ideal for various scenarios:

### Personal Knowledge Management
Build a personal library of web resources for research, learning, or reference. The tagging and list features make it easy to create a structured knowledge base.

### Team Collaboration
Share curated link collections with team members. Multiple user support allows teams to collaboratively build bookmark collections.

### Research Projects
Organize sources and references for academic or professional research. The archive feature ensures that referenced content remains accessible even if the original source disappears.

### Content Curation
Create themed collections of web content for specific topics or interests. The guest access feature allows sharing these collections publicly.

## Getting Started

To begin using LinkAce:

1. **Choose your deployment method** - Docker is recommended for beginners
2. **Install LinkAce** following the official documentation
3. **Create your account** and configure basic settings
4. **Import existing bookmarks** if migrating from another service
5. **Install browser extensions** or bookmarklets for easy bookmark saving
6. **Start organizing** with lists and tags

## Advantages of Self-Hosting

Choosing LinkAce as a self-hosted solution offers several benefits:

- **Data Privacy**: Your bookmarks remain on your own server

- **Customization**: Modify the application to suit your specific needs
- **No Service Limits**: Store unlimited bookmarks without subscription tiers
- **Longevity**: Not dependent on a company's business model or continued operation
